Programming Projects

These projects are simple Java programs I created while learning the basics of programming. They helped me understand how code works, how to think logically, and how to fix mistakes when something didn't run properly.

Project 1: Basic Java Practice Program

Goal of the project:
This was a small practice program where I used Java to print messages, do simple calculations, and experiment with variables. The main purpose was to get familiar with writing code, running it, and understanding how Java syntax works.

My development process:
I started by writing a very basic program using System.out.println to print text. After that, I added variables and tried doing calculations with numbers. At first I got a few errors because I forgot semicolons, but fixing those helped me understand how strict Java is with syntax.

What I learned:
I learned how to write simple Java statements, how variables work, and how to fix small errors. This project helped me get more comfortable reading error messages and understanding what the computer needed.

Project 2: Workout Tracker Java Program (Practice)

Goal of the project:
This was a practice program I created to simulate a workout tracker. The idea was to ask the user for the number of workouts completed in a week, store the value, and then print a simple message based on their progress.

My development process:
I used Scanner to take user input, stored the number in a variable, and added a few if statements to print different responses. I had trouble at first because I forgot to import the Scanner class, but after adding import java.util.Scanner;, the program worked. I also practiced running the program multiple times to test different inputs.

What I learned:
This project helped me understand how user input works in Java, how to use conditional statements, and why testing code with different values is important. Even though the program was simple, it showed me how logic and user interaction fit together.
