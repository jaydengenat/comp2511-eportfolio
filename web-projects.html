Web Projects

These projects are small websites I built while learning HTML and CSS. They are simple, but they helped me understand how to structure a page, connect files, and use basic styling to make a layout look more organized.

Project 1: NBA Top 5 Players Page

What I was asked to do / trying to do:
I created a basic webpage that listed my top five NBA players. Each player had their name, team, and a short description. The goal was to practice building a clean page with headings, paragraphs, and a simple layout.

My development process:
At first, I just wrote everything in plain text, but then I organized it using different heading levels and sections so it was easier to read. I also added a list for the players and tried styling the page with CSS to center the content and adjust the spacing. I had a few small issues with my CSS not applying, which I later realised was because the file path to my CSS file was wrong. Fixing that helped me understand how linking files works.

What I learned:
From this project I learned how basic HTML structure works (headings, paragraphs, lists), how to link a CSS file, and how to use simple properties like margins, text alignment, and background color. Even though the page was simple, it made HTML and CSS feel a lot more familiar.

Project 2: Fitness Tips Webpage

What I was trying to do:
On my own time I made this project, I created a simple webpage that shared a few basic fitness tips and exercises. The goal of the page was to practice organizing information into clear sections and improving the overall layout using beginner-level HTML and CSS.

My development process:
I started by planning out the sections I wanted: a short introduction, a list of tips, and a small section explaining why staying active is important. Once I wrote the HTML, I added CSS to style the headings, add spacing, and make the page look cleaner. I also practiced using lists to organize the fitness tips. At first, some of my CSS didn't apply correctly, but I realized it was because I forgot to save the file. After saving and refreshing, everything worked.

What I learned:
This project helped me understand how useful simple HTML structures like lists and sections can be when organizing information. I also got more comfortable adjusting spacing, font sizes, and page layout with CSS. Overall, this helped me build cleaner, more readable pages.
